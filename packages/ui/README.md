# @youversion/platform-react-native-expo-ui

YouVersion Platform Bible components for React Native (Expo): `BibleReader`, `BibleCard`, `BibleTextView`, `VerseOfTheDay`, theming, and native bottom sheets. Built on the [React Web SDK](https://github.com/youversion/platform-sdk-react) via [Expo DOM Components](https://docs.expo.dev/guides/dom-components/).

Pairs with `@youversion/platform-react-native-expo-core` (installed alongside).

## Install

```bash
npx expo install @youversion/platform-react-native-expo-ui @youversion/platform-react-native-expo-core
```

Requires Expo SDK 56+, React 19, and a [dev build](https://docs.expo.dev/develop/development-builds/introduction/) (not Expo Go). Peer dependencies are listed in `package.json`.
